# AI 模型选择器 - 用户指南（免费版 v1.0）

## 🎯 这是什么？

AI 模型选择器是一个智能工具，帮助你根据任务需求选择最合适的 AI 模型，并预估成本和耗时。

## ✨ 核心功能

1. **智能任务分析** - 自动识别任务类型、复杂度、技术要求
2. **Token 3-tier 预估** - 提供 Min/Avg/Max 三档预估
3. **成本预估** - 根据实时价格计算使用成本
4. **3-tier 模型推荐** - 免费首选 / 性价比之选 / 效果最佳
5. **价格动态更新** - 支持自动/每小时/每天/手动更新

## 📦 快速开始

### 安装

```bash
# 克隆或下载本项目
cd "Smart Task-Model Selector"

# 安装依赖（如果需要）
npm install
```

### 使用

#### 方式 1：命令行交互

```bash
node index.js
```

然后按提示输入任务描述，例如：
- "帮我用 Python 写一个爬虫"
- "分析这份财报，提取关键指标"
- "写一篇关于 AI 未来发展的深度文章"

#### 方式 2：API 调用

```javascript
const AIModelSelector = require('./index.js');

const selector = new AIModelSelector();

async function main() {
  const result = await selector.selectModel(
    "帮我用 Python 写一个爬虫",
    {
      accuracyMode: 'standard',     // 'standard' | 'high'
      optimizationPreference: 'balanced',  // 'cost' | 'balanced' | 'quality'
      updateMode: 'auto'            // 'auto' | 'hourly' | 'daily' | 'manual'
    }
  );
  
  console.log(result.display);
  
  // 处理用户选择
  const choiceResult = await selector.handleUserChoice(1, result);
  console.log(choiceResult);
}

main();
```

## 🎛️ 配置选项

### 预估精度（accuracyMode）

| 模式 | 说明 | 误差范围 | 速度 |
|------|------|----------|------|
| `standard` | 标准精度 | ±30% | 快 |
| `high` | 高精度 | ±10% | 较慢 |

### 优化偏好（optimizationPreference）

| 偏好 | 说明 | 适用场景 |
|--------|------|----------|
| `cost` | 省钱优先 | 预算敏感、简单任务 |
| `balanced` | 平衡模式 | 大多数场景（推荐） |
| `quality` | 效果优先 | 复杂任务、高质量要求 |

### 价格更新模式（updateMode）

| 模式 | 说明 | 适用场景 |
|------|------|----------|
| `auto` | 智能更新（推荐） | 缓存 < 1小时：使用缓存；1-24小时：警告；> 24小时：强制刷新 |
| `hourly` | 每小时更新 | 成本敏感用户 |
| `daily` | 每天更新 | 普通用户（推荐） |
| `manual` | 手动更新 | 完全控制更新时机 |

## 📊 输出说明

### 1. 价格数据状态

显示价格数据的更新时间、时效性、来源，并提供刷新选项。

### 2. 任务分析结果

- **任务类型**：代码生成 / 文案写作 / 搜索研究 等
- **复杂度**：简单 / 中等 / 复杂
- **技术要求**：代码、示例、模板 等
- **置信度**：⭐ (1-5星）

### 3. Token 消耗预估

| 预估级别 | 说明 | 适用场景 |
|---------|------|----------|
| 🟢 最小 | 理想情况，一次过 | 简单任务、有类似经验 |
| 🟡 平均 | 标准情况，正常交互 | 大多数场景 |
| 🔴 最大 | 复杂情况，多轮调试 | 复杂任务、新技术 |

### 4. 模型推荐对比

#### 🆓 免费首选
- **特点**：完全免费，适合预算敏感用户
- **推荐场景**：学习、测试、简单任务

#### 💰 性价比之选
- **特点**：成本适中，效果良好
- **推荐场景**：大多数生产环境（推荐）

#### 💎 效果最佳
- **特点**：效果最好，成本较高
- **推荐场景**：复杂任务、高质量要求

### 5. 模型成本对比表

显示所有候选模型的：
- 单价（/1K tokens）
- 预估成本
- 效果评分（⭐ 1-5星）

### 6. 智能建议

根据任务需求和预算，提供选择建议。

## 🔄 高级功能

```javascript
const result = await selector.selectModel(
  "写一个爬虫",
  {
    forcePriceRefresh: true  // 强制刷新价格
```

```javascript
const result = await selector.selectModel(

  }
);
```
