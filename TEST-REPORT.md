### 🐛 已修复的 Bug

1. **价格时效显示 "20582 天前"** ✅ 已修复
   - 原因：`cacheAge` 计算错误（当 `lastUpdate` 为 `null` 时）
   - 修复：添加 `null` 检查，首次加载时显示 "刚刚"

2. **Token 预估显示 "undefined"** ✅ 已修复
   - 原因：`formatEstimate` 参数名不匹配（`cost_per_1k` vs `cost_per_1k_tokens`）
   - 修复：统一参数名为 `cost_per_1k`，并兼容两种写法

3. **输出重复显示** ✅ 已修复
   - 原因：`generateFullDisplay` 调用 `generateTokenEstimateSection` 时传参错误
   - 修复：正确传递 `tokenDisplay.tokens` 和 `tokenDisplay.cost`

## 已知限制

1. **文件缓存未实现**
   - 当前使用内存缓存，重启后丢失
   - 建议：实现基于文件的缓存（~/.smart-task-model-selector/cache.json）

2. **OpenRouter API 集成未实现**
   - 当前使用模拟数据
   - 建议：实现真实 API 调用

3. **模型数据库硬编码**
   - 当前在代码中定义模型列表
   - 建议：从 `config/models.yaml` 加载

4. **历史数据优化未实现**
   - 当前不使用历史数据调整预估
   - 建议：记录实际使用数据，优化预估精度
