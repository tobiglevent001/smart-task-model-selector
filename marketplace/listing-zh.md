# Smart Task-Model Selector
## 智能任务模型选择器

**为你的任务智能匹配最合适的AI模型**

---

## 核心功能

- 🧠 **智能任务分析** - 理解代码生成、文案写作、翻译、研究分析等多种任务
- 🎯 **精准模型匹配** - 根据任务类型、复杂度、技术要求推荐最优模型
- 💰 **双币种成本显示** - 同时显示人民币 ¥ 和美元 $
- 🌍 **国内外模型支持** - 通义千问、DeepSeek、OpenAI、Anthropic 等
- ⚡ **成本优化建议** - 帮你找到性价比最高的方案

## 快速开始

```
/smart-task-model-selector
```

用自然语言描述你的任务，即可获得最佳模型推荐！

## 支持的任务类型

| 类型 | 推荐模型 | 适用场景 |
|------|---------|---------|
| 代码生成 | 通义千问、DeepSeek Coder、Claude | 网站开发、小程序、API |
| 文案写作 | Qwen Long、Kimi、GPT-4o-mini | 文章、报告、方案 |
| 翻译 | DeepL、Google翻译、百度翻译 | 多语言内容 |
| 搜索研究 | Perplexity、DeepSeek搜索 | 新闻分析、市场调研 |
| 数据分析 | Claude 3 Opus、GPT-4o | 图表、报告、洞察 |

## 成本示例

典型1000 Token响应：

- 省钱：¥0.005（DeepSeek V3）
- 高端：$0.01（Claude 3.5 Sonnet）
