# Smart Task-Model Selector

**Automatically match your task to the best AI model with cost intelligence**

---

## Features

- 🧠 **Smart Task Analysis** - Understands code generation, writing, translation, research, and more
- 🎯 **Intelligent Model Matching** - Recommends optimal models based on task type, complexity, and technical requirements
- 💰 **Dual Currency Cost Display** - Shows costs in both ¥CNY and $USD
- 🌍 **Domestic & International Models** - Supports Alibaba, DeepSeek, OpenAI, Anthropic, and more
- ⚡ **Budget Optimization** - Helps you find the most cost-effective solution

## Quick Start

```
/smart-task-model-selector
```

Describe your task in natural language, and get instant model recommendations!

## Supported Task Types

| Category | Models | Best For |
|----------|--------|----------|
| Code Generation | Qwen Coder, DeepSeek Coder, Claude 3.5 Sonnet | Website, app development |
| Writing | Qwen Long, Kimi, GPT-4o-mini | Articles, reports, proposals |
| Translation | DeepL, Google Translate, Baidu | Multi-language content |
| Research | Perplexity, DeepSeek Search | News analysis, market research |
| Data Analysis | Claude 3 Opus, GPT-4o | Charts, reports, insights |

## Cost Example

For a typical 1000-token response:
- Free: ¥0 (Qwen Coder)
- Budget: ¥0.005 (DeepSeek V3)
- Premium: $0.01 (Claude 3.5 Sonnet)
