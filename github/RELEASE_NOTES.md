# v1.0.0 发布说明

## 🎉 Smart Task-Model Selector 正式发布！

这是智能任务模型选择器的首个正式版本。

### 核心功能

1. **智能任务分析** - 自动识别代码生成、文案写作、翻译、搜索研究等任务类型
2. **精准模型推荐** - 根据任务复杂度、技术要求、用户偏好推荐最优模型
3. **双币种成本显示** - 同时显示人民币和美元价格，方便比较
4. **区域偏好支持** - 支持国内优先/国际优先模式

### 支持的模型

**国内平台**
- 通义千问 (Qwen)
- DeepSeek
- Kimi
- 百度文心
- 讯飞星火

**国际平台**
- OpenAI (GPT-4o, GPT-4o-mini)
- Anthropic (Claude 3.5 Sonnet, Claude 3 Opus)
- Perplexity
- DeepL

### 如何使用

在 CodeBuddy 中输入：
```
/smart-task-model-selector
```

然后描述你的任务，即可获得最佳模型推荐！
